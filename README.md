There are total 5 games which are clones of basic old games made with simple mechanics and UI.

It requires sfml library of version 2.6

the minimun c++ version is C23.


### HOW TO BUILD
1. From root directory type this -  cmake -S . -B build
     (it should be done properly if all the dependencies and correcty version are installed)
   
2. From the same directtory -  cmake --build build


## How to run the final game-
1. go into build folder - cd build
2. there must be 5 folders for all 5 games after the build is completed
     - breakout
     - space_invaders
     - pong
     - platformer
     - snake

3. Go into the respective game sub folder
4. run the executable by typing "./" before the game name
     eg- inside /cgrind/build/snake
                  ./snake
